# INVAIZ

인바이즈의 시그니처 서체로, 특별한 경우에만 사용합니다. 인바이즈의 제품 이름이나 일부 제목을 강조할 때 사용됩니다.

export const TypefaceInvaiz = styled.p`
  ${({ theme }) => theme.style.fontFace.invaiz};
  ${({ theme, textStyle }) => theme.font.invaiz[textStyle]};
`;

export function InvaizTextStyles() {
  return (
    <>
      {Object.keys(font.invaiz).map((textStyle) => (
        <TypefaceInvaiz textStyle={textStyle} key={textStyle}>
          {textStyle}
        </TypefaceInvaiz>
      ))}
    </>
  );
}

# Noto Sans

export const TypefaceNotoSans = styled.p`
  ${({ theme }) => theme.style.fontFace.notoSans};
  ${({ theme, textStyle }) => theme.font.notoSans[textStyle]};
`;

export function NotoSansTextStyles() {
  return (
    <>
      {Object.keys(font.notoSans).map((textStyle) => (
        <TypefaceNotoSans textStyle={textStyle} key={textStyle}>
          {textStyle}
        </TypefaceNotoSans>

  );
}

## Emphasis Text

제목 또는 본문에 강조할 부분이 생겼을 경우, 색상 또는 두께를 다르게 적용하는 등 변화를 줄 수 있습니다.
이 경우에도 일정한 규칙에 따라 변화를 적용시켜야 합니다.

export const Point1 = styled.span`
  ${({ theme }) => theme.font.emphasisText.point1}
`;

export const Point2 = styled.span`
  ${({ theme }) => theme.font.emphasisText.point2}
`;

<Canvas>
  <Story name='Emphasis Text'>

    </p>
    <p>
      오늘 반찬은 <Point2>제육볶음</Point2>입니다.
    </p>
  </Story>
</Canvas>

## No Input Text

중요도가 낮은 부가 정보를 기입하거나, 입력해야 하는 텍스트란이 공백일 때 텍스트를 약하게 처리합니다.
텍스트를 약하게 처리할 때는 다음과 같은 규칙을 따릅니다.

export const NoInputText = styled.p`
  ${({ theme }) => theme.font.noInputText};
`;

## Text Link

export const TextLink = styled.span`
